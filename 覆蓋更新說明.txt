PackPilot v0.4.0 免開發環境更新方式

1. 先關閉 GitHub Desktop 以外正在開啟專案檔案的程式。
2. 將本壓縮檔解壓縮。
3. 把解壓縮後的所有檔案複製到你原本的 packpilot-new 專案資料夾。
4. 選擇「取代目的地中的檔案」。
5. 不要刪除原專案內的 .git 資料夾。
6. 開啟 GitHub Desktop，確認 Changes 中出現修改。
7. Summary 填入：Update PackPilot v0.4.0
8. 按 Commit to main，再按 Push origin。
9. 到 GitHub 網站的 Actions，等待部署出現綠色勾勾。
10. 若網頁仍顯示舊版，先用無痕視窗開啟，或清除該網站的 Service Worker／網站資料。

網站：
https://kimogksj.github.io/packpilot-new/

重要：
- 本版固定使用 /packpilot-new/。
- 本版不需要你執行 npm install 或 npm run build。
- GitHub Actions 會在 Push 後自動安裝與建置。
