# PackPilot v0.4.0

## 新增
- 每個通路改為多階段流水線，撿貨、分貨、包貨可同時進行。
- 每個階段可指定不同執行者、不同計時方式與手動時間。
- 單一工作與單一階段皆可修改。
- 工作可複製、取消／封存、恢復與永久刪除。
- 階段可略過，並保留修改歷程。
- 中斷原因與恢復紀錄。

## 調整
- 作業時間只計算撿貨、分貨、包貨與庫存系統使用。
- 等待寄貨與等待搬運不計入總作業時間或效率。
- 寄貨與搬運保留獨立階段，但不納入作業效率。
- 固定 GitHub Pages 路徑為 `/packpilot-new/`。
- 全站固定黑色深色模式。

## 資料相容
- LocalStorage schema 升級至 v4。
- 會嘗試將 v0.3 舊工作資料轉換為新的階段格式。
