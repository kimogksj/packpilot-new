# PackPilot Product Principles

1. Real work flow takes priority over theoretical flow.
2. The home screen shows what needs attention next before what is already finished.
3. Anything that can be timed must be reportable.
4. Operational actions must have a safe way back.
5. The screen should carry context so the worker does not need to remember it.
6. Automatic tracking is the default; manual editing is for correction.
7. Editing data must not destroy history.
8. During a workday, the present task has higher visual priority than completed history.
