PackPilot v0.6.0 Alpha Revision 2.1 覆蓋更新方式

1. 解壓縮本 ZIP。
2. 若要覆蓋既有 GitHub 專案，保留原本的 .git 資料夾。
3. 其餘檔案以本版本全部覆蓋。
4. 使用 GitHub Desktop Commit，再 Push origin。
5. 等待 GitHub Actions 完成部署。

本版本沿用 packpilot-data-v6-alpha-r1 儲存位置，並將 schema 8 或 9 自動遷移到 schema 10，既有 Alpha 資料會保留。

主要修正：
- 撿貨、分貨、包貨、搬到走廊在完成後都會顯示「復原完成」。
- 復原後回到該流程完成前的狀態。
- 原本正在計時的流程會從按下復原的時刻重新接續，不會把已完成期間算進工時。
- 復原包貨時，系統會一併撤回自動開始的等待物流狀態。
- 復原搬到走廊時，已完成的宅配工作會重新回到進行中。
